# ✈️ Flight Booking Chatbot

An interactive AI chatbot that helps users explore and book flights through a simple conversational interface.

---

## 🚀 Live Demo

👉 https://huggingface.co/spaces/kav096/flight-booking-chatbot

---


## 📌 Features

* 💬 Chat-based UI
* ✈️ Flight booking assistance
* ⚡ Fast Gradio interface
* 🧠 Simple intent-based responses

---

## 🛠️ Tech Stack

* Python
* Gradio
* Hugging Face Spaces

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
```
